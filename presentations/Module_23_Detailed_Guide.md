# MODULE 23
ADAS Capstone: Calibration & Validation

## Goal
Evaluate ADAS implementation quality through profile tuning, scenario testing, and safety justification.

## What Students Do
1. Select a base profile (conservative/balanced/aggressive).
2. Run scenarios and collect metrics (warning events, critical events, brake commands).
3. Tune values and compare outcomes.
4. Present policy decisions and mitigation plan.

## Instructor Notes
- Grade ADAS outcomes, safety reasoning, and validation process.
- Coding skill is optional and not required for success.
